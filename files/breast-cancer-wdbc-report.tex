\section{Dataset}
The Wisconsin Diagnostic Breast Cancer (WDBC) dataset contains 569 samples with 30 numeric features derived from
fine needle aspirate images. The target label indicates malignant (M) or benign (B) tumors.

\section{Preprocessing}
We removed the identifier column, encoded diagnosis as binary (M=1, B=0), and used stratified train/test splits.
Summary statistics are shown below.

\subsection{Data Overview}
\begin{itemize}
  \item Rows: 569
  \item Columns: 32
  \item Missing values: 0
  \item Duplicate rows: 0
\end{itemize}

\section{Modeling and Evaluation}
